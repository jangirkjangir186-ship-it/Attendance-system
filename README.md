# Smart Attendance System

##  Description

Smart Attendance System is a modern project designed to replace manual attendance methods. It uses QR codes or basic face recognition to automatically mark and record attendance, making the process faster and more efficient.

## Features

* QR code-based attendance
* Basic face recognition
* Automatic attendance recording
* Time-saving and error reduction

##  Technologies Used

* HTML
* CSS
* JavaScript (or Java/Python for face recognition)

## How It Works

Users scan a QR code or use face recognition to mark their attendance. The system automatically records the data with time and date, reducing manual work and improving accuracy.

##  Why This Project?

* Solves real-world problem
* Modern and innovative idea
* Useful for schools, colleges, and offices


 This project makes attendance tracking smart, fast, and reliable.
